# Wikipedia Dump Parsers

A variety of parsers for extracting information from Wikipedia dump files. 

## Dependencies
* mwxml

## Usage
python main.py filename titles namespace
* filename: the wikipedia dump file, which is normally downloaded from Wikipedia's dump site (e.g., https://dumps.wikimedia.org/enwiki/20190301/enwiki-20190201-pages-meta-history1.xml-p10p2066.7z). The filename has .7z extension but it should be unzipped
